# nop-plugin-payments-pagarme-creditcard

Pagar.me plugin para o Nop.Commerce.
<br/>

Suporta pagamento via cartão de crédito e estorno integral do pagamento.
<br/>
Faz checagem posterior do pagamento via `Schedule Tasks`.

## Instalação

Copiar os arquivos da pasta `dist/Payments.PagarMeCreditCard` para a pasta do servidor `Plugins/Payments.PagarMeCreditCard`.

No módulo de plugins (`Configuration/Plugins/Local Plugins`) clicar no botão `Reload list of plugins`. O novo plugin deverá aparecer. Clique em `Install`. 

## Configuração do Plugin

No módulo de pagamentos (`Configuration/Payment/Payment Methods`) clique na opção `Configure` do plugin. 
Na próxima tela informar a `Api Key`e a `Crypt Key`fornecidas pelo Pagar.me.
<br/>

De volta a listagem habilite o plugin.
